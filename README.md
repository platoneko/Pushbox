# Teamwork Pushbox

## Usage

使用logisim打开TeamWork_v2.0.circ，需要加载cs3410.jar。

从res/pushbox_data_v2.0.hex加载数据到TeamWork_v2.0.circ中的数据存储器，启动时钟（务必要等地图加载完毕才能进行操作），将所有箱子推到目的地。
